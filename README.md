## О проекте

Сегментация фона на видеопотоке с возможностью выбора фона и стилизации персональной информации пользователя.

## Установка

```
git clone https://github.com/703lovelost/t1-hackathon-nsk-2025.git
cd t1-hackathon-nsk-2025
npm install
```

## Запуск

```
npm start
```
